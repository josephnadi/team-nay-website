# Team Website

Initial project structure for the team website.

## Getting Started

### Prerequisites
- A web browser
- Git installed on your machine

### Installation
To get a local copy up and running, follow these simple steps:

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/team-nay.git
   ```
2. Navigate to the project directory
   ```bash
   cd team-nay-website
   ```
3. Open `index.html` in your browser to view the site.

## Project Structure
- `index.html`: Main entry point
- `css/`: Stylesheets
- `js/`: JavaScript files
- `images/`: Image assets
