# Team Nay Website

Initial project structure for the team website.

## Getting Started

### Prerequisites
- A web browser
- Git installed on your machine

### Installation
To get a local copy up and running, follow these simple steps:

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/team-nay.git
   ```
2. Navigate to the project directory
   ```bash
   cd team-nay-website
   ```
3. Open `index.html` in your browser to view the site.

## Project Structure
- `index.html`: Main entry point
- `css/`: Stylesheets
- `js/`: JavaScript files
- `images/`: Image assets



# Project Description
This project is a collaborative multi-page website created as part of a Git and GitHub team assignment. The goal of the project is to practice version control, teamwork, branching, and pull request workflows.

## Features
- Home page with basic navigation
- About page with team information
- Services/Projects page
- Contact page with form validation

## Installation / Running the Project
1. Clone the repository
2. Open the project folder in VS Code
3. Open `index.html` in a web browser
